## v0.7.1（2026-07-23）

### 新增
- 新增多份普通备份合并导入：可一次选择多个普通 ZIP 备份，合并导入其中的任务、图片、收藏夹、Agent 对话和 API 配置；分片备份仍需一次选中同一批次的全部分片，且不能与普通备份混选。

### 变更
- 对项目代码进行了大规模重构，优化模块划分和状态管理，并补充测试覆盖，提升后续维护和功能迭代效率。
- 改进参考图编辑流程：点击参考图统一打开大图预览，并在预览中直接替换图片或进入遮罩编辑；移除原「参考图编辑按钮」偏好设置，同时优化预览背景、移动端显示尺寸、翻页按钮和图片计数位置。
- 改进遮罩编辑器：优化顶部操作区和深色模式样式，笔刷大小支持在整条竖向滑杆区域点击或拖动调节。
- 将新建或切换至 Responses API 配置时使用的默认模型更新为 `gpt-5.6-sol`。

### 修复
- 修复按 README 配置 Vercel 自动更新后，同步上游 Release 不会触发自动部署的问题 (#122)。
