## v0.6.11（2026-06-30）

### 新增
- **支持在默认 API URL 中携带默认配置参数**：`VITE_DEFAULT_API_URL` 现在可以通过 URL 参数设置默认 API 配置，包括 `apiUrl`、`apiKey`、`apiMode`、`model`、`profileName`、`codexCli`、`streamImages` 和 `streamPartialImages`，便于预置部署环境的默认模型与连接方式。

### 变更与优化
- **优化默认 API 配置初始化逻辑**：统一复用流式中间图像数量的归一化逻辑，并让默认配置中的 `streamImages` 与 API 模式默认值保持一致，减少部署参数组合下的配置偏差。
- **优化 Agent 模型配置下拉菜单展示效果**：悬浮或长按 Agent 模型配置下拉菜单中的选项时，会显示该配置项的全部内容，便于用户快速了解和选择。

### 修复
- **修复 Agent 模式图片任务断线后无法正确恢复的问题**：Agent 调用 fal.ai 或自定义异步图像任务时，如果连接中断，会保留队列任务信息并自动查询结果；恢复成功后会补全工具输出并继续 Agent 轮次，恢复失败时会把错误写回对应工具调用。
